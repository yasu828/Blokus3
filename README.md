## 実装できた機能
- ブロックをおく盤面の作成
- 各々の形のブロックの作成
- 所持したブロックを盤面にドラッグ&ドロップ

## これから実装する機能
- ブロックを4人分作成
- 選択したブロックを回転、反転させる
- ブロック同士でドラッグ&ドロップできないようにする
- gridareaで各々の配置を整える
- flexboxで見た目調整
- ヘッダーの表示

## 徐々につけていきたい機能

- 最終プレイヤーがpathを選択時ゲーム終了する
- ゲーム終了時、勝敗を集計する
- ドロップしたブロックが盤面からはみ出る時の処理
